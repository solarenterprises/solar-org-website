Solar Website 

Requirements:
- nodejs (v18)
- pm2
- yarn


```
git clone https://github.com/solar-network/website
cd website
yarn install
node solar-website.js
```

for live editing use "pm2 start solar-website.js --watch"

```
pm2 start solar-website.js --watch
```
Website visible at "https://localhost:8000"
